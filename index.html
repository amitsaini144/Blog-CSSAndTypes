CSS and it's Types

Introduction:

CSS stands for cascading style sheets. CSS is used to design the website.When we have made the page by using HTML after that we need to design it so that our website can look attractive and interactive. It makes a website more user friendly.
By using faces we can change the colour ,we can change the size of the font, we can change the background colour and much more.
So lets get into our basic CSS.

3 ways to add CSS into our web pages are as :

1) Inline CSS
2) Internal CSS
3) External CSS

Among above 3 ways,best way is adding CSS is External CSS.

1) Inline:

Style attribute is used to add styling. Basically in this type of CSS we add style attribute into the tag which we want to design.

Example :suppose we want to change the color of text inside h1 tag.



2) Internal:

In this type of CSS we do styling inside the head tag instead of writing style tag into an attribute individually which was very hectic. We can style multiple attributes at once by various ways like by using class name , element name and id name.

Example:Suppose we want to change the colour of whole body to red.



3) External CSS:

This is the best way out of the three ways. We do styling in other page which we save it as filename.css. We link this page to our HTML page in which we want import the styling. This way is very useful when we want to style more than one page.
